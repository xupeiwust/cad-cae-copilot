# Handoff Benchmark Results

This directory contains completed benchmark run records for the agent handoff benchmark (Phase 14D).

Each run is recorded as a Markdown file (`run_YYYYMMDDTHHMMSSZ.md`) filled from `result_template.md`, and optionally as a JSON file (`run_YYYYMMDDTHHMMSSZ.json`) conforming to `results.schema.json`.

No benchmark runs have been conducted yet for the Phase 14D scaffold.
