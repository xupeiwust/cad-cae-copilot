# Real Bracket Benchmark Run Package (Phase 11C)

This directory is the manual benchmark scaffold for the real-geometry bracket scenario.

It compares two conditions:

- **Condition A (raw STEP):** AI reads only `examples/real_bracket.step` (or an equivalent copied file).
- **Condition B (`.aieng` package):** AI reads resources generated by `scripts/run_real_step_demo.py`, including topology, AAG, feature graph, constraints, simulation setup, protected regions, summaries, and validation status.

## Execution Records

- **`results_run_001.md`**: Phase 11D Run 001 — documented blocked execution due to unavailable optional OCP/CadQuery backend; recorded package generation success but noted backend unavailability.
- **`results_run_002.md`**: Phase 11D Run 002 — successful end-to-end execution with OCP/CadQuery backend available; package generation and validation confirmed; 549/549 tests pass.
- **`results_ai_run_001.md`**: Phase 11E AI benchmark record comparing raw real STEP versus curated .aieng semantic package responses.

Phase 11E note: real STEP scored higher than previous mock raw STEP on geometry usefulness, but the curated .aieng package still provided full CAX semantic usefulness.
